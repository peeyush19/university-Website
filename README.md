# 🎓 University Website (HTML Project)

A simple and responsive **University Website** created using pure **HTML**.  
This project is designed for beginners to understand the basics of web development using only HTML without CSS frameworks or JavaScript.

## 📌 Features

- 🏠 **Home Page**
  - Navigation bar to move between pages
  - University image/photo section
  - Information about the university

- 📝 **Student Registration Form**
  - Student details form with required fields
  - Input validation using HTML form attributes

- 🔐 **Login Page**
  - Username and password login form

- 📚 **Courses Page**
  - Table displaying available university courses
  - Includes course name, duration, and course details

## 🛠️ Technologies Used

- HTML5

## 📂 Project Structure

```bash
University-Website/
│
├── index.html              # Home Page
├── registration.html       # Student Registration Form
├── login.html              # Login Page
├── courses.html            # Courses Page
└── README.md               # Project Documentation
````

   OR
